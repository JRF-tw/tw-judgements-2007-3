臺灣彰化地方法院刑事判決　　　　　　　96年度易緝字第20號
公　訴　人　臺灣彰化地方法院檢察署檢察官
被　　　告　乙○○
上列被告因偽造文書案件，經檢察官提起公訴（92年度偵字第97
10號、93年度偵字第8889號），本院判決如下：
    主  文
乙○○共同明知為不實之事項，而使公務員登載於職務上所掌之
公文書，足以生損害於公眾，處拘役伍拾陸日，如易科罰金，以
銀元佰元即新臺幣玖佰元折算壹日。
　　犯罪事實
一、緣許秀清於民國87年間，以結婚為由，要求當時之同居人甲
    ○○將其所有、坐落於彰化縣彰化市○○段下小段第236
    之14地號土地及其上建物，及坐落彰化縣鹿港鎮○○段第19
    67之1 地號、第1967之4 地號土地應有部分7 分之1 悉數贈
    與許秀清，甲○○與許秀清二人並於87年5 月18日，假以買
    賣為登記原因，辦理所有權移轉登記（該二人已另行審結）
    。嗣許秀清取得上開房地所有權後，無意婚嫁，為免甲○○
    反悔索回上開房地，遂與表弟乙○○共同謀議，是時，乙○
    ○、許秀清明知二人間並無債權債務關係，卻基於共同犯意
    聯絡，由許秀清委請不知情之代書張芳惠代為辦理上開房地
    之抵押權設定事宜 (但申請文件上代理人為張芳惠之助理王
    秀梅)，而分別於87年6月19日、25日，設定抵押權新台幣（
    下同）5百萬元及1百萬元，使彰化縣彰化地政事務所公務員
    於公務所掌之公文書上登載不實，均足生損害於該地政事務
    所公務登載之正確性。
二、案經甲○○訴請及臺灣彰化地方法院檢察署檢察官自動檢舉
    偵查起訴。
    理  由
壹、證據能力：
    本案下列引用之供述及非供述證據，經本院依法踐行調查證
    據程序，公訴人及被告均不爭執各該證據之證據能力，且亦
    查無依法應排除其證據能力之情形，是後述所引用證據之證
    據能力均無疑義，合先敘明。
貳、認定犯罪事實所憑之證據及理由
一、訊據被告乙○○矢口否認犯行，辯稱：伊與許秀清之間確有
    債權債務關係，且前揭房地設定抵押權一事，伊均不知情，
    亦未參與，全然為許秀清一人所為云云。
二、然查：
（一）被告乙○○與另一被告許秀清確有設定前揭抵押權之事實
      ，有該建物、土地之登記謄本及設定文件在卷可稽。
（二）又該抵押權所擔保之債權債務關係是否存在，厥為本案審
      究之重點。茲核：
      另一被告許秀清先表示該抵押權擔保之債務係因向被告
        乙○○借錢來發給其所經營之良公司員工之工資云云
         (見臺灣彰化地方法院91年度訴字第890號民事卷92 年
        1月9日言詞辯論筆錄)。於本院審理時許秀清卻陳稱：
        係因開公司要擴廠，需要增資，所以就開始陸續向乙○
        ○借錢，用來支付公司擴廠所需要的土地租金、廠房、
        機器設備、原料等開銷，這些大概須2百多萬。之後又
        因公司營運不好，開予廠商的票陸續跳票，怕這些廠商
        把票軋進去，便情商將票拿回來以一比一的比例兌換現
        金，因而再向乙○○借錢來換票，共計約5百萬元云云
        （見本院94年度易字第83號院卷二影卷第94頁背面）。
　　　許秀清又表示係自85年起向被告乙○○借款，由被告乙
        ○○林口農會活期帳戶轉帳予許秀清，計620 餘萬元，
        此據許秀清提出93年8 月18日陳報狀、收據8 紙、存提
        款存摺明細影本1 份為證，且許秀清表示向被告乙○○
        借得之款項，係先轉帳至乙○○之父錢金池林口農會甲
        存帳戶後，再由錢金池開支票予許秀清，許秀清收受後
        即轉讓予其他債權人，此為另一被告許秀清於93年8 月
        30日陳報狀中載明。惟許秀清所稱之前開支票，實係錢
        金池用來購買飼料及簽發予友人楊茂琳，且楊茂琳並不
        認識許秀清，與許秀清亦無債權債務關係乙節，亦據證
        人王愛珠、楊茂琳於偵查中證述無訛（見臺灣彰化地方
        法院92年度偵字第9710號卷第135-137 頁），並有錢金
        池林口農會帳號000000000000000號甲存帳戶之交易明
        細表、支票影本17張可查。甚且，許秀清陳稱錢金池之
        支票是用來支付伊養豬飼料之款項，然卻不認識支票持
        有人楊茂琳及黃榮杰云云 (見另一被告許秀清於93年11
        月12日之偵訊筆錄）。
      依上所述，許秀清所陳稱借款之原因、憑據及證明互有
        出入，是否真實顯有疑問。況許秀清另證稱：確實欠被
        告乙○○約本金750 萬元，但兩次抵押權登記之債權金
        額係由許秀清自己一人決定，乙○○並沒有提供意見，
        且設定總計600 萬元之抵押權金額係因許秀清外公有一
        些遺產，被告乙○○要求許秀清之母放棄遺產繼承權，
        即可少算債款金額，至於少算多少，乙○○並不知情，
        ...... ， 設定前並未徵求被告乙○○之同意，待許秀
        清之母快過世時，即約設定完2 年才向被告乙○○表示
        ，有將房地設定抵押權予被告乙○○，共600 萬云云（
        見上開本院94年度易字第83號院卷二影卷第94頁），則
        許秀清既係自行決定設定金額，衡情，上開房地為許秀
        清所有，其上之負擔越少越好，何以須設定高達600 萬
        元之債權額？又設定後遲至2 年之久方告知債權人有此
        擔保，亦與常情相違背。綜上所述，上開抵押權所擔保
        之債權明顯為假。
（三）又查代為辦理本案抵押權設定登記之代書張芳惠已於另案
      審理時到庭結稱「（問：關於抵押權的設定是何人委託你
      辦理？）答：是許秀清及其父母。在辦完過戶之後，來辦
      理的，對方是一個錢先生，是他們的親戚，我有問是否確
      實有債權債務關係存在，他們說確實有，我請乙○○來，
      但他說工作不方便，所以我用電話照會他，他有把相關證
      件寄下來，是寄給我們還是許秀清那邊我忘記了」等語（
      見上開本院94年度易字第83號院卷二影卷第102 頁），核
      與證人即許秀清之父許建致到庭結稱「（問：是否記得去
      芳惠那裡設定抵押權的情形？）答：設定給乙○○，我也
      有和我太太一起去。現場還有許秀清，乙○○沒有去，代
      書和乙○○聯絡，之後就設定好了」等語中所述「代書確
      有照會乙○○」一情悉相合致。且抵押權登記申請文件中
      有檢附1 份乙○○之戶口名簿影本，此有該份申請文件在
      卷可查（見上開本院94年度易字第83號院卷一影卷第123-
      124頁），就此，據被告乙○○供陳：曾提供1份有伊及父
      母、祖母、小孩共約9人之全戶戶口名簿影印予伊姑媽即
      許秀清之母，之後就沒有在提供予伊姑媽了云云（見本院
      審判筆錄第8頁），然核上開申請文件中所附之戶口名簿
      根本並非上述被告乙○○所稱有約9人在戶內之戶口名簿
      ，而係戶內僅有被告乙○○及其妻二人之戶口名簿，是以
      該申請文件中所附之戶口名簿顯非被告乙○○曾提供予他
      人（即許秀清之母）者，則若非被告乙○○自行提供，何
      以會出現在該份申請文件內？由此益徵上揭證人張芳惠所
      證：有照會乙○○，乙○○有寄相關證件下來等情確屬實
      在，足以採信，則被告乙○○知情並參與而寄送相關證件
      之事實堪以認定，被告乙○○辯稱並不知情、亦未參與云
      云要屬飾卸之詞，不足憑採。
（四）被告乙○○、許秀清明知假債權仍故為設定，其等犯罪事
      證自屬明確，應依法論科。
、論罪科刑理由
一、比較新舊法
    被告行為後，刑法於94年2 月2 日修正公布，並於95年7 月
    1 日施行，關於本案應適用之新、舊法，茲論述如下：
（一）刑法第2條第1項規定，係規範行為後法律變更所生新舊法
      律比較適用之準據法，於95年7月1日刑法施行後，應適用
      刑法第2條第1項之規定，為「從舊從輕」之比較（最高法
      院95年5月23日第8次刑事庭會議決議第1點第1項參照）。
      但如新舊法處罰之輕重相同，即無比較適用之問題，非此
      條所指之法律有變更，即無本條之適用，應依一般法律適
      用原則，適用裁判時法（最高法院95年11月7日第21 次刑
      事庭會議決議參照）。
（二）依刑法第2條第1項規定為新舊刑法比較時，應就罪刑有關
      之共犯、未遂犯、連續犯、牽連犯以及累犯加重、自首減
      輕、暨其他法定加減原因、與加減例等一切情形，綜其全
      部之結果，而為比較，再適用有利於行為人之法律處斷，
      而不得一部割裂分別適用不同之新舊法（最高法院24年上
      字第4634號判例意旨、上開95年第8 次及第21次刑事庭會
      議決議參照）。
（三）依上原則，經比較如附表後，以被告行為時之舊法較有利
      於被告，依刑法第2 條第1 項前段規定，該部分應一體適
      用被告行為時之法律即修正前刑法第33條第5 款、罰金罰
      鍰提高標準條例第1 條前段、現行法規所定貨幣單位折算
      新臺幣標準第2 條之規定。
（四）至於刑法第28條有關共同正犯之規定，因僅作文字修正，
      對於狹義共同正犯（指有犯意聯絡及行為分擔之數行為人
       ）之認定，不生任何影響，並非刑法第2 條第1 項所指
      之法律有變更，並無刑法第2 條第1 項之適用，應依一般
      法律適用原則，適用裁判時法。
（五）易科罰金之折算標準，新法施行後，應依新法第2 條第1
      項之規定，適用最有利於行為人之法律（最高法院95年5
      月23 日 第8 次刑事庭會議決議第3 點第2 項參照）。本
      案就易罰金部分，經比較如附表後，以被告行為時之舊法
      較有利於被告，依上說明，易科罰金部分應適用被告行為
      時之法律即修正前刑法第41條第1 項前段、修正前罰金罰
      鍰提高標準條例第2 條之規定。
二、核被告乙○○所為，係犯刑法第214 條之罪。被告乙○○與
    另一被告許秀清就虛偽設定抵押權登記部分，有犯意聯絡及
    行為分擔，為共同正犯。被告乙○○、許秀清利用不知情之
    代書為假債權設定抵押權登記，為間接正犯。
三、爰審酌被告乙○○否認犯行，顯無悔意，及所犯雖影響地政
    機關公務登載之正確性，惟除此之外，所造成之損害尚非重
    大，及其犯罪之動機、目的、犯罪時所受之刺激、犯罪之手
    段、犯罪後之態度等一切情狀，量處如主文所示之刑，並依
    修正前刑法之規定諭知易科罰金之折算標準。
肆、適用法條依據
（一）應依刑事訴訟法第299條第1項前段。
（二）刑法第28條、第214 條、（修正前）第41條第1 項前段。
（三）罰金罰鍰提高標準條例第1條前段、（修正前）第2條。
本案經檢察官高如應到庭執行職務。

│ 2  │易科罰金│被告行為後，刑法第41條第1項前段關於易科罰金折算標準之規 │
│    │        │定，業於94年2月2日修正公布，並於95年7月1日施行，由「犯最│
│    │        │重本刑為五年以下有期徒刑以下之刑之罪，而受六個月以下有期│
│    │        │徒刑或拘役之宣告，因身體、教育、職業、家庭之關係或其他正│
│    │        │當事由，執行顯有困難者，得以一元以上三元以下折算一日，易│
│    │        │科罰金。」修正為「犯最重本刑為五年以下有期徒刑以下之刑之│
│    │        │罪，而受六個月以下有期徒刑或拘役之宣告者，得以新臺幣一千│
│    │        │元、二千元或三千元折算一日，易科罰金。」應依刑法第2條第 │
│    │        │1項規定，適用最有利於行為人之法律（最高法院95年第8次刑事│
│    │        │庭會議決議第3點第2項參照）。而修正前刑法第41條第1項前段 │
│    │        │之易科罰金折算標準，依修正刪除前之罰金罰鍰提高標準條例第│
│    │        │2條規定，應就其原定數額提高為100倍折算1日，並依現行法規 │
│    │        │所定貨幣單位折算新臺幣條例第2條規定，以新臺幣元3倍折算之│
│    │        │，亦即修正前刑法第41條第1項前段之易科罰金折算標準，應以 │
│    │        │ 銀元300元即新臺幣900元折算為1日，經比較新、舊法結果，應│
│    │        │以被告行為時之法律舊法較有利於被告，依刑法第2條第1項前段│
│    │        │規定，是本案自應適用被告行為時之法律即修正前刑法第41條第│
│    │        │1項前段、修正前罰金罰鍰提高標準條例第2條規定，併諭知易科│
│    │        │罰金之折算標準。                                        │

論罪科刑法條
中華民國刑法第214條
(使公務員登載不實罪)
明知為不實之事項，而使公務員登載於職務上所掌之公文書，足
以生損害於公眾或他人者，處3年以下有期徒刑、拘役或500元以
下罰金。
