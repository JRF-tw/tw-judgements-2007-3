臺灣桃園地方法院刑事判決　　　　　　 96年度訴緝字第129號
公　訴　人　臺灣桃園地方法院檢察署檢察官
被　　　告　乙○○原名林天信.
上列被告因偽造文書案件，經檢察官提起公訴（94年度偵字第13
582 號），被告於準備程序中就被訴事實為有罪之陳述，經本院
合議庭裁定由受命法官獨任行簡式審判程序，茲判決如下：
    主  文
乙○○共同連續行使偽造私文書，足以生損害於公眾及他人，處
有期徒刑柒月。如附表一所示之「應沒收之物」均沒收。
    事  實
一、乙○○、謝垣廣（通緝中）與徐文忠（業經本院以95年訴字
    第733 號判處有期徒刑1 年4 月在案）共組頂贊企業有限公
    司（下稱頂贊公司，名義負責人為黃智，即丙○○之夫，業
    已死亡），謝垣廣為實際負責人、徐文忠為總經理，乙○○
    則負責車輛調度，呂紹賢（未據起訴）為公司股東，而謝垣
    廣與大陸籍人士顧愈芳（業已於94年9 月29日遣返而通緝中
    ）為姘居男女朋友。於民國93年10月間，適逢甲○○欲將其
    所出資購買、登記於其子羅仕康名下之位於桃園縣中壢市○
    里○○鄰○村街106 號2 樓之房地（下稱該房地）出賣，其男
    友呂紹賢乃藉詞介紹其與徐文忠、謝垣廣接洽，徐文忠、謝
    垣廣、乙○○、顧愈芳、呂紹賢五人因頂贊公司欠缺資金，
    欲詐取不法利益，乃共同基於意圖為自己不法所有、行使偽
    造私文書、行使使公務員登載不實之公文書之概括犯意聯絡
    ，於93年10月間某日，在該房地址，由呂紹賢在場慫恿，謝
    垣廣、徐文忠向甲○○佯稱欲幫其代為高價出售該房地云云
    ，甲○○誤以為渠等確係欲為其出售該房地而陷於錯誤，遂
    將該房地之所有權狀、羅仕康之印鑑章、印鑑證明（於93年
    11月8 日向戶政機關申請）等資料交予徐文忠、謝垣廣以委
    託渠等出賣該房地。嗣後上開五人乃先後為下列行為分擔，
    並利用不知情之護照代辦人員曹淑真、代書事務所人員謝佩
    珊為相關手續之代辦：
  (一)首推由乙○○於93年10月間某日在台北縣泰山鄉丙○○租屋
    處向其不知情之甥女丙○○以其父死亡後遺產須辦理繼承為
    由，詐得丙○○之身分證後，再交予謝垣廣；謝垣廣乃將丙
    ○○身分證及顧愈芳所提供之照片轉交予姓名年籍不詳之人
    ，以新台幣（下同）2 萬元之代價取得上貼有顧愈芳照片、
    記載丙○○年籍資料之變造身分證（見附表一編號一所示）
    ；且因向銀行貸款需以雙證件申請帳戶為撥款之用，乃偽以
    丙○○名義簽立「中華民國普通護照申請書」（見附表一編
    號二），於93年11月3 日利用不知情之護照代辦人員曹淑真
    持以向外交部領事事務局行使該變造身份證以申請護照，該
    局未察覺有異，乃於93年11月5 日核發上貼有顧愈芳照片、
    記載丙○○年籍資料之中華民國護照一本（見附表一編號三
    ）；另又於不詳時間偽造丙○○印章一枚（見附表一編號四
    ）、偽以「桃順企業社」及負責人「楊順山」名義開立丙○
    ○「在職服務證明書」一份（見附表一編號五）。
  (二)嗣謝垣廣將上開變造之丙○○身分證影本、以上開方式取得
    之護照影本、偽造之丙○○印章、「在職服務證明書」交予
    徐文忠，徐文忠乃連同甲○○所交付之上開資料，交予不知
    情之溫智謀代書事務所（由職員謝佩珊承辦），並利用委託
    不知情之代書事務所人員謝佩珊誤以為業經丙○○、羅仕康
    之授權而以其等名義代為繕寫「不動產買賣契約書」（見附
    表編號六），並持蓋用上開偽造之丙○○印章、羅仕康之印
    章；於93年12月1 日向桃園縣中壢地政事務所辦理該房地之
    所有權移轉事宜，並提出代為繕寫之「土地登記申請書（移
    轉所有權）」（見附表一編號七）、「土地、建築改良物買
    賣所有權移轉契約書」（見附表一編號八）、該房地桃園縣
    政府稅捐稽徵處房屋稅繳款書等資料向地政機關辦理該房地
    之所有權移轉事宜以行使該等偽造私文書、變造身分證，使
    地政事務所承辦之公務員誤以為真而登載於所執掌之公文書
    上。
  (三)謝佩珊又於93年12月17日持上開資料交予板信商業銀行桃園
    分行（下稱板信商銀）聯絡辦理買賣房屋之抵押貸款事宜以
    行使，經板信商銀通知對保後，徐文忠乃於93年12月23日載
    顧愈芳至板信商銀對保，顧愈芳遂偽稱其為丙○○，徐文忠
    為免日後遭查獲乃偽稱其為乙○○，並以丙○○名義與板信
    商銀簽立「板信商業銀行個人不動產借款契約書」（見附表
    一編號九）、「土地、建築改良物抵押權設定契約」（見附
    表一編號十）、「承諾書」（見附表一編號十一）、「委託
    代扣約定書」（見附表一編號十二）、「綜合同意書」（見
    附表一編號十三）、「切結書」（見附表一編號十四）、「
    收據」（見附表編號十五）等資料，另填寫「板信銀行存摺
    存款開戶申請書」（見附表一編號十五）以辦理該行帳戶供
    日後撥款匯入之用（帳號0000-0 00-0000000） ，由板信商
    銀承辦人員張志雄對保後誤以為真，經板信商銀承辦人員審
    核後陷於錯誤核准授信貸予新台幣（下同）250 萬元。
  (四)後經謝佩珊持上開資料至桃園縣中壢地政事務所辦理設定抵
    押權，並以丙○○名義代為繕寫「土地登記申請書（設定抵
    押權）」（見附表一編號十七）辦理抵押權設定，使地政事
    務所承辦人員誤以為真而登載於所執掌之公文書。
    渠等五人上開所為均足生損害於地政機關對於土地登記資料
    管理之正確性、楊順山、桃順企業社、丙○○、羅仕康、甲
    ○○、板信商銀。嗣板信商銀乃於93年12月31日撥款250 萬
    元至上開帳戶，嗣並將該帳戶存摺交予徐文忠，渠等並於94
    年1 月5 日偽造丙○○受羅仕康委託「委託書」（見附表一
    編號十九）向安泰商業銀行中壢分行以行使用以領取「清償
    證明」，並委以丙○○名義簽立「簽收單」（見附表一編號
    十八）；嗣徐文忠隨即於94年1 月6 日持該帳戶金融卡詐領
    現金746,000 元（該帳戶扣除清償前手貸款後僅餘746058元
    ）得手後，由謝垣廣分別交予呂紹賢及挪做公司資金使用。
    嗣因上開貸款之分期本息均未給付，甲○○收受板信商銀於
    94年3 月21日寄發至該房地址之催告書，復輾轉聯丙○○始
    知上情，並於94年4 月12日聯袂至台灣桃園地方法院檢察署
    提出告訴。
二、案經甲○○、丙○○訴由台灣桃園地方法院檢察署檢察官偵
    查起訴
    理  由
一、本件被告乙○○所犯係死刑、無期徒刑、最輕本刑為三年以
    上有期徒刑以外之罪，其於準備程序中就被訴事實為有罪之
    陳述，經法官告知簡式審判程序之旨，並聽取公訴人、被告
    之意見後，依刑事訴訟法第273 條之1 第1 項規定，經本院
    合議庭裁定，由受命法官獨任進行簡式審判程序，並為審理
    判決，合先敘明。
二、上開犯罪事實，業據被告乙○○於本院審理時坦承不諱，核
    與共犯謝垣廣之供述（警詢、偵訊）相符，並有共犯顧愈芳
    於警、偵訊時之供述、證人甲○○之證述（警詢、偵訊、本
    院）、證人丙○○之證述（警詢、偵訊）、共犯呂紹賢之證
    述（偵訊、本院）、證人謝易男、張志雄、謝佩珊之證述（
    本院）在卷；此外，另有板信商銀雙掛號之催告書（見偵卷
    第49至50頁）、附表一編號一至十九所示之證據資料、板信
    商銀借款申請書暨客戶基本資料表、個人戶授信批覆書（見
    偵卷第53至54頁）、頂贊公司、桃順企業社營利事業登記基
    本資料查詢（見本院卷一第102 至1106頁）、板信商銀丙○
    ○帳戶0000-000-0000000號交易明細表（見偵卷第205 頁）
    、安泰商銀羅仕康帳戶交易明細表、財團法人金融聯合徵信
    中心函覆甲○○、羅仕康授信之料（見本院卷一第57至58頁
    、本院卷三第52頁至63頁）、市話00-0000000、00-0000000
    、00-0000000號、行動電話0000000000號申請人資料（見本
    院卷二第34、89至91頁）、外交部領事事務局95年8 月16日
    領一字第09552315800 號函覆（見本院卷二第49頁）、桃園
    縣平鎮市戶政事務所95年12月1 日桃平戶字第0950007758號
    函覆羅仕康印鑑證明申請書（見本院卷三第18至23頁）等資
    料在卷可稽，被告乙○○上開犯行足堪認定。
三、論罪科刑
  (一)按行為後法律有變更者，適用行為時之法律。但行為後之法
    律有利於行為人者，適用最有利於行為人之法律，94年2 月
    2 日修正公佈，並自95年7 月1 日施行之刑法（下稱新法；
    修正前刑法下稱舊法）第2 條第1 項定有明文，此項規定，
    係規範行為後法律變更所生新舊法律比較適用之準據法，於
    新法施行後，自應適用此規定，為「從舊從輕」之比較。查
    被告行為後，其所犯下列各罪之刑法規定雖未修正，惟其中
    詐欺得利罪、變造特種文書罪、行使使公務員登載不實罪之
    條文之法定本刑均有罰金刑，而刑法罰金刑之最低數額、加
    減規定亦均有如附表一所示之修正，且刑法業已刪除連續犯
    、牽連犯之規定，自應依新法第2 條第1 項規定，為「從舊
    從輕」之比較，並綜合全部罪刑之結果比較新舊法如附表二
    所示（即應適用整體是用修正前之法律）。
  (二)核被告林天信所為，均係犯刑法第339 條第2 項詐欺得利罪
    、同法第216 、212 條之行使變造特種文書罪、同法第216
    、210 條之行使偽造私文書罪、同法第216 、214 條行使使
    公務員登載不實罪。又偽造署名仍係偽造私文書之階段行為
    ，又偽造私文書、變造特種文書、使公務員登載不實之低度
    行為應為行使偽造私文書、行使變造特種文書及行使使公務
    員登載不實之高度行為所吸收，均不另論罪，應各依行使偽
    造私文書、行使變造特種文書及行使使公務員登載不實罪論
    處。另被告乙○○與謝垣廣、徐文忠、呂紹賢、顧愈芳就上
    開犯行，有犯意聯絡及行為分擔，為共同正犯。又渠等利用
    不知情之護照代辦人員、代書人員為上開行為之部分，為間
    接正犯。被告乙○○上開所為多次行使變造特種文書、偽造
    私文書、使公務員登載不實之文書，其時間緊接，各自所犯
    構成要件相同之罪名，顯係基於概括犯意而為之，各為連續
    犯，各應依修正前刑法第56條規定論以一罪，並加重其刑。
    被告乙○○所犯上開罪名間，有方法結果之牽連關係，應依
    修正前刑法第55條之規定，從一重之連續行使偽造私文書罪
    論處。爰審酌被告乙○○，夥同其餘四人為上開犯行，多次
    偽造文書，所生危害頗為嚴重，惟犯後坦承犯行，深表悔悟
    ，另參考其素行、智識程度、生活狀況等一切情狀，量處如
    主文所示之刑。
  (三)又公訴人雖未就被告乙○○等人多次行使該變造特種文書、
    偽造私文書、使公務員登載不實之公文書之部分提起公訴，
    惟此部分犯罪事實與已起訴之犯罪部分，有連續犯之裁判上
    一罪關係，業如前述，為起訴效力所及，本院自得併予審理
    。
  (四)至如附表一編號二、四至十九所示之「應沒收之物」，為被
    告乙○○等人偽造之「丙○○」印章、印文、署押，均應依
    刑法第219 條予以沒收；而附表一編號一、三、二十、二十
    一所示之「應沒收之物」，則為被告乙○○等人所有、用以
    犯本件犯罪或因本件犯罪所得之物，均應依刑法第38條第1
    項第2 款予以沒收。至於該等偽造之印文、署押所附著之文
    件，均業已交予銀行等行使，已非被告乙○○等人所有之物
    ，自無從予以宣告沒收，末此敘明。
四、按本條例施行前，經通緝而未於中華民國九十六年十二月三
    十一日以前自動歸案接受偵查、審判或執行者，不得依本條
    例減刑，中華民國九十六年罪犯減刑條例第5 條定有明文；
    查被告前經本院於95年5 月12日發佈通緝，於96年7 月16日
    17時40分許始經緝獲到案，有桃園縣政府警察局中壢分局通
    緝（協尋）案件移送書一份在卷可稽，是被告係於該條例施
    行前（即96年7 月16日）遭通緝，而於該條例施行後緝獲到
    案，既非自動歸案接受審判，依上開規定，自不得依該條例
    減刑，末此敘明。
據上論斷，應依刑事訴訟法第273 條之1 第1 項、第299 條第1
項前段，刑法第2 條第1 項前段、第210 條、第212 條、第214
條、第216 條、第339 條第2 項、第219 條、第38條第1 項第2
款，修正前刑法第28條、第56條、第55條前段，罰金罰鍰提高標
準條例第1 條前段，判決如主文。
本案經檢察官鄭東峰到庭執行職務。

附錄論罪科刑法條
中華民國刑法第210 條（偽造變造私文書罪）
偽造、變造私文書，足以生損害於公眾或他人者，處5 年以下有
期徒刑。
中華民國刑法第212 條（偽造變造特種文書罪）
偽造、變造護照、旅券、免許證、特許證及關於品行、能力服務

│    │                      │                      │            │  ；就減輕規定而言│
│    │                      │                      │            │  ，因新法就罰金之│
│    │                      │                      │            │  最低度亦予減輕，│
│    │                      │                      │            │  舊法則無，比較結│
│    │                      │                      │            │  果新法對被告較為│
│    │                      │                      │            │  有利，為免法律割│
│    │                      │                      │            │  裂適用，經綜合全│
│    │                      │                      │            │  部罪刑比較結果，│
│    │                      │                      │            │  仍應適用舊法（詳│
│    │                      │                      │            │  如後述）。      │
│    │                      │                      │            │                  │
├──┴───────────┴───────────┴──────┴─────────┤
│結論：為免法律割裂適用，比較時應就罪刑有關之共犯、未遂犯、想像競合犯、牽連犯、連續犯、│
│結合犯，以及累犯加重、自首減輕暨其他法定加減原因（如身份加減）與加減例等一切情形，綜其│
│全部罪刑之結果而為比較（最高法院95年度第8 次刑事庭會議決議一 (四)參 照），故本案經綜合│
│全部罪刑比較結果，以上新法並非較有利於被告，應適用舊法第33條第5 款、第56條、第68條，決│
│定本件法定罰金刑之最低數額、加減，並論以連續犯、牽連犯。                              │
